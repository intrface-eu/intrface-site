# Funda Product Page PRD

## Page Name
Funda Product Page

## Goal
Present Funda as a funding opportunity platform that helps organizations discover, evaluate, and act on relevant funding opportunities through AI-assisted workflows, role-based collaboration, and geospatial matching.

## Primary Audience
- Organizations seeking funding opportunities
- Consultants helping clients pursue funding
- Public/private funding ecosystem stakeholders
- Technical evaluators reviewing Intrface flagship products

## Secondary Audience
- Admin and director stakeholders
- Visitors exploring the Intrface portfolio

## Key Message
Funda is an AI-powered funding opportunity platform that combines discovery, matching, collaboration, and application workflow support for organizations, consultants, and administrators.

## Page Job To Be Done
The page should help visitors understand:
1. what Funda is
2. who it serves
3. how it improves funding discovery and execution
4. what role-based workflows exist in the system
5. why it is more than a searchable directory

## Required Sections
- Hero
- What Funda solves
- Audience / role-based workflow model
- Core capabilities
- AI-assisted matching and workspace flow
- Platform proof / architecture summary
- Final CTA

## Required Proof
- Corporate, consultant, and admin roles represented accurately
- AI-assisted opportunity matching and workspace flow reflected honestly
- No invented claims unsupported by README or repo reality

## Required CTAs
- Explore Funda
- Get in Touch
- View Platform Overview

## Must-Avoid Misrepresentations
- Do not frame Funda as only a grant search engine
- Do not omit the collaborative workflow / application workspace dimension
- Do not overclaim automation beyond documented platform behavior

## Content Model Needs
- overview
- role summaries
- feature highlights
- workflow steps
- CTA links

## Design Notes
- Use Google Sans Flex as the primary typeface
- Funda should feel clear, trustworthy, and structured
- Prioritize readability and confidence over visual experimentation

## Acceptance Criteria
- Visitors understand Funda is a workflow platform, not just a listings database
- Role-specific value is visible
- Core features are grounded in actual product functionality
